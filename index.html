Fluidsimulationsolver

Smooth Particle Hydrodynamics solver written in C++ and OpenGL using OpenMP framework for parallel processing and spatial hashing technique for neighbour search.

Download this project as a .zip file Download this project as a tar.gz file

For this project, Lagrangian particle-based approach has been considered and hence implemented along with Smooth Particle Hydro Dynamics for generating fluid simulation. The proceeding section highlights this approach and how it differs from Eularian approach to solve Navier-Stokes equation of fluid.
